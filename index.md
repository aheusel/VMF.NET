# VMF.NET API Documentation

VMF.NET is a lightweight modeling framework for .NET that translates annotated C# interfaces into powerful implementations via a Roslyn Source Generator.

## Packages

| Package | Description |
|---|---|
| VMF.NET.Runtime | Public interfaces, VList, ChangesManager |
| VMF.NET.Json | System.Text.Json serialization and JSON Schema generation |
